  - 유효한 `RandomSource.nextInt(n)`은 `0 <= value < n` 정수여야 한다. 위반 값은 modulo나
    clamp 없이 `RangeError`로 거부해 잘못된 `GameState`를 만들지 않는다.

    바꾸는 것은 SG-010 `step` 책임이다.

    두 독립 test fixtures를 주입해 증명했다.

  - `src/ui/contracts.ts` mirror가 계속 존재한다. SG-007 public types와 드리프트할 위험은 현재

    실제 domain import 전환을 수행해야 한다.

    한다. 이번 PASS는 AC-G01~G10 전체 완료를 주장하지 않는다.
  - 지정 Claude 독립 리뷰와 통합은 남아 있다. 이 세션은 자기 결과를 verified/merged로 올리지
    않았다.
